PROGRAM

This program aims to provide visulistations of socialeconomical indicators within different Seattle GEOID's. Compare and contrast some of these indicators and analyse some of there distributions.It is a key focus that lots of these indicators be compared geospatialy and the analysis provides key information about the quality of fitting of distributions.

-MATPLOTLIB (matplotlib.pyplot)
-PANDAS
-SEABORN
